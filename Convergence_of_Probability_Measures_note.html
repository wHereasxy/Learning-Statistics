§1. Measures on Metric Spaces / 度量空间上的测度

1.1 正则性与积分表示

Original (Cleaned)

We begin by studying probability measures on the general metric space. Denote the space by S, and let S be the Borel σ-field, the one generated by the open sets; its elements are the Borel sets. A probability measure on S is a nonnegative, countably additive set function P satisfying PS = 1.

If probability measures Pn and P satisfy Pn f → P f for every bounded, continuous real function f on S, we say that Pn converges weakly to P and write Pn ⇒ P.

中文翻译

我们首先研究一般度量空间上的概率测度。记该空间为 S，并令 S 为 Borel σ-域（Borel σ-field），即由开集生成的σ-域；其元素称为Borel集。一个概率测度是 S 上的非负、可数可加的集合函数 P，满足 PS = 1。

如果概率测度 Pn 和 P 满足：对每一个 S 上的有界连续实函数 f，都有 Pn f → P f，则称 Pn 弱收敛（converges weakly）于 P，记为 Pn ⇒ P。

直观理解

弱收敛的直觉: 将概率测度视为一种"质量分布"。弱收敛 Pn ⇒ P 意味着，对于任何"好的"（有界连续的）检验函数 f，用 Pn 对 f 积分的结果收敛到用 P 对 f 积分的结果。这比"逐点收敛"弱，但比"集合测度收敛"强。

类比: 如果在实数线上 Fn(x) → F(x) 对所有 F 的连续点 x 成立，则对应的概率测度弱收敛。这正是经典概率论中学生熟悉的"依分布收敛"概念——弱收敛是它在度量空间上的推广。

Theorem 1.1 — 正则性 (Regularity)

Every probability measure P on (S, S) is regular: for every S-set A and every ε > 0, there exist a closed set F and an open set G such that F ⊂ A ⊂ G and P(G − F) < ε.

中文: (S, S) 上的每个概率测度 P 都是正则（regular）的：对每个Borel集 A 和 ε > 0，存在闭集 F 和开集 G 使 F ⊂ A ⊂ G 且 P(G − F) < ε。

通俗解释

正则性说的是：任何Borel集 A 都可以被一个闭子集和一个开超集"夹逼"，且两者的测度差可以任意小。这等价于说：

P(A) = sup{ P(F) : F closed, F ⊂ A } = inf{ P(G) : G open, A ⊂ G }

为什么重要? 正则性意味着概率测度完全由它在闭集（或开集）上的值决定。这是后续 Portmanteau 定理（用开集/闭集的测度刻画弱收敛）的理论基础。

Theorem 1.2 — 连续函数决定测度

Probability measures P and Q on S coincide if Pf = Qf for all bounded, uniformly continuous real functions f.

中文: 如果对有界一致连续的实函数 f 有 Pf = Qf，则 S 上的概率测度 P 和 Q 相同。

证明思路

证明的核心是构造一个特殊的连续函数来逼近示性函数 IF：令 f(x) = (1 − ρ(x, F)/ε)+。这个函数是：

  • 有界的: 取值在 [0, 1]
  • 一致连续的: |f(x) − f(y)| ≤ ρ(x,y)/ε
  • 满足: IF(x) ≤ f(x) ≤ IFε(x)

然后通过取极限 ε↓0 可得 PF ≤ QF 和 QF ≤ PF（对闭集F），再用定理1.1的正则性推出 P = Q。

1.2 紧性 (Tightness) 与四个关键例子空间

定义: Tightness / 紧性

A probability measure P on (S, S) is tight if for each ε > 0 there exists a compact set K such that PK > 1 − ε.

中文: 对每个 ε > 0，存在紧集 K 使得 PK > 1 − ε。

直观理解: Tightness

紧性 (tightness) 说的是: 概率质量不会"逃逸到无穷远"。几乎所有的概率质量都可以被某个紧集（在度量空间中，"紧"大致等同于"有界闭"加上某种完备性条件）所覆盖。

反例: 在实数线上，令 Pn = δn（在点 n 的单位质量）。对任何紧集 K（在 ℝ 中就是有界闭集），当 n 足够大时，n ∉ K，故 PnK = 0。这个序列不是 tight 的——质量"逃逸到正无穷"了。

Theorem 1.3 (完可分空间上的紧性)

If S is separable and complete, then each probability measure on (S, S) is tight.

中文: 如果 S 是可分（separable）且完备（complete）的（即Polish空间），则每个概率测度自动是紧的。

概念定义在概率论中的作用

完备性 (Complete)每个Cauchy列都收敛保证极限存在不"逃出"空间；对Prohorov定理至关重要

1.3 分离类 (Separating Class)

定义: Separating Class / 分离类

A subclass A of S is a separating class if two probability measures that agree on A necessarily agree also on the whole of S.

中文: 如果两个概率测度在集合类 A 上一致，则它们在全体 Borel 集上也必然一致。

分离类的例子
  • 闭集全体: 由定理1.1（正则性）可知是分离类
  • ℝk: 形如 (−∞, x]（即 (−∞, x₁] × ... × (−∞, xₖ]）的集合构成分离类 — 这正是在经典概率论中"分布函数决定测度"的推广
  • ℝ∞: 有限维集合（finite-dimensional sets） — 形如 πk−1H，H ∈ Rk — 构成分离类（甚至是 π-系统）
  • C[0,1]: 同样地，有限维集合 πt₁...tₖ−1H 构成分离类

注意: 分离类和收敛决定类（convergence-determining class）是不同的概念！分离类保证"同一个极限测度"被唯一确定，收敛决定类则要求测度序列的弱收敛可以通过该类集合上的收敛来判定。

§2. Properties of Weak Convergence / 弱收敛的性质

2.1 The Portmanteau Theorem / Portmanteau定理

Theorem 2.1 — The Portmanteau Theorem

Let Pn, P be probability measures on (S, S). The following five conditions are equivalent:

  1. Pn ⇒ P (weak convergence by definition)
  2. Pn f → P f for all bounded, uniformly continuous f
  3. lim supn Pn F ≤ P F for all closed F
  4. lim infn Pn G ≥ P G for all open G
  5. Pn A → P A for all P-continuity sets A (i.e., P(∂A) = 0)
中文要点

Portmanteau 定理给出了弱收敛的 五个等价刻画，任何一条都可以用作弱收敛的定义：

  1. 基本定义: 对所有有界连续 f，积分收敛
  2. 减弱: 只需对一致连续的 f 成立
  3. 闭集版本: 闭集的上极限 ≤ 极限测度的闭集概率 — "质量不会突然涌入闭集"
  4. 开集版本: 开集的下极限 ≥ 极限测度的开集概率 — "质量不会突然从开集中流失"
  5. 连续集版本: 对边界测度为零的集合（P-连续集），测度逐点收敛
Portmanteau定理的具体例子

例1 (点质量的弱收敛): 在 ℝ 上，令 xn = 1/n 且 x₀ = 0。则 δ1/n ⇒ δ₀（δx 是在点 x 的单位质量）。

  • 对闭集 F = {0}: lim sup PnF = 0 ≤ 1 = P F ✓ — 注意到这里是不等式，是严格的小于！
  • 对开集 G = ℝ \ {0}: lim inf PnG = 1 ≥ 0 = P G ✓ — 同样是严格的大于！
  • 对集合 A = {0}: PnA = 0 ↛ 1 = P A — 但 ∂{0} = {0} 且 P(∂A) = 1 ≠ 0，所以 A 不是 P-连续集，这不违反条件(v)!

例2 (离散逼近连续分布): 将 [0,1] 分成 n 个等长子区间，在每个子区间中点放质量 1/n。则有理节点上的经验分布弱收敛到 [0,1] 上的均匀分布（Lebesgue测度）。这本质上是 Riemann 积分的概率论表达。

Portmanteau定理的证明思路

证明的链条是: (i) ⇒ (ii) 显然; (ii) ⇒ (iii) 使用定理1.2中的近似函数 f(x) = (1 − ρ(x,F)/ε)+; (iii) ⇔ (iv) 通过补集运算; (iii)&(iv) ⇒ (v) 利用 A° ⊂ A ⊂ A⁻ 及边界条件; (v) ⇒ (i) 利用分层表示 Pf = ∫₀¹ P[f > t] dt。

关键点: 条件(iii)和(iv)合起来给出了"夹逼"效应: P A° ≤ lim inf Pn A ≤ lim sup Pn A ≤ P A⁻。当 P(∂A) = 0 时，两端的界限相等，因此中间收敛。

2.2 收敛决定类 (Convergence-Determining Class)

收敛决定类 vs 分离类

A class A is convergence-determining if, for every P and every sequence {Pn}, convergence PnA → P A for all P-continuity sets in A implies Pn ⇒ P.

中文: 收敛决定类要求对 A 中所有 P-连续集上的收敛能推出整个序列的弱收敛。

关键例子与反例

收敛决定类的例子:

  • ℝk: 矩形 [a₁,b₁]×...×[aₖ,bₖ] 构成收敛决定类（定理2.4）。因此，"分布函数逐点收敛于连续点"是弱收敛的等价刻画
  • ℝ∞: 有限维集合构成收敛决定类（例2.4）。这意味着：ℝ∞ 上的弱收敛等价于所有有限维分布的弱收敛

关键反例 — C[0,1] 中的有限维集合不是收敛决定类!（例2.5）

考虑函数 zn(t) = nt·1[0,1/n](t) + (2 − nt)·1(1/n,2/n](t)。zn → 0 逐点但不一致收敛。令 Pn = δzn 和 P = δ₀，则 Pn ⇏ P（因为 zn 不一致收敛到 0）。但对于任何有限维投影 πt₁...tₖ，当 n 充分大时（2/n 小于最小的非零 tᵢ），πt₁...tₖ(zn) = πt₁...tₖ(0) = (0,...,0)。因此所有有限维分布都收敛，但序列本身不弱收敛！

这个例子说明：在 C[0,1] 空间中，弱收敛理论本质上超越了有限维理论 — 这是第2章所有技术的出发点。

2.3 连续映射定理 (The Mapping Theorem)

Theorem 2.7 — The Mapping Theorem

Let h: S → S′ be measurable S/S′ and let Dh be the set of its discontinuities. If Pn ⇒ P and P Dh = 0, then Pn h−1 ⇒ P h−1.

中文: 如果 h 在极限测度 P 下几乎处处连续（即其不连续点集的 P-测度为零），则映射后的测度也弱收敛。

连续映射定理的应用实例

例 — 分位数变换 (Example 2.8): 令 F 是分布函数，ϕ(u) = inf{x: u ≤ F(x)} 是其分位数函数（quantile function）。则：

Lebesgue 测度 P 通过 ϕ 的像测度 Pϕ−1 正是具有分布函数 F 的概率测度。

如果通过[0,1]上的均匀经验分布来逼近 P（如通过小数展开生成近似均匀的观测），则由连续映射定理可生成近似具有分布 F 的观测序列。

例 — ℝ∞ 的有限维投影: 投影 πk 是连续的，故 Pn ⇒ P 推出有限维分布收敛。在 ℝ∞ 中逆命题也成立（有限维集合是收敛决定类），但在 C 中不成立。

§3. Convergence in Distribution / 依分布收敛

Original (Cleaned)

Let X be a mapping from a probability space (Ω, F, P) to a metric space S. We call X a random element if it is measurable F/S. ... We say a sequence {Xn} of random elements converges in distribution to the random element X if Pn ⇒ P, where Pn and P are the distributions of Xn and X. In this case we write Xn ⇒ X.

Thus Xn ⇒ X if and only if E[f(Xn)] → E[f(X)] for all bounded, continuous f on S.

中文翻译

令 X 是从概率空间 (Ω, F, P) 到度量空间 S 的映射。如果 X 是 F/S-可测的，则称 X 为 随机元（random element）。其分布（distribution）是 P = P X−1。如果 Pn ⇒ P（其中 Pn 是 Xn 的分布），则称 Xn 依分布收敛（converges in distribution）于 X，记作 Xn ⇒ X。

术语统一
空间 S随机元的名称
ℝ¹随机变量 (random variable)
ℝk随机向量 (random vector)
ℝ∞随机序列 (random sequence)
C[0,1], D[0,1] 等函数空间随机函数 (random function)
重要: Xn 和 X 可以定义在不同的概率空间上!

依分布收敛只关心分布（即 S 上的像测度）是否收敛，而不关心底层概率空间 (Ω, F, P) 的结构。这是弱收敛理论的巨大优势——我们可以自由地在任何方便的概率空间上构造随机元。

3.1 依概率收敛 (Convergence in Probability)

Convergence in Probability to a Constant

Xn converges in probability to a if P[ρ(Xn, a) ≥ ε] → 0 for each ε > 0. We write Xn ⇒ a (this is consistent with δa being the limit).

中文: 对每个 ε > 0，有 P[ρ(Xn, a) ≥ ε] → 0。

Theorem 3.1 & 3.2 — Slutsky-type Results

Thm 3.1: If Xn ⇒ X and ρ(Xn, Yn) ⇒ 0, then Yn ⇒ X.
Thm 3.2: If Xun ⇒n Zu and Zu ⇒u X, and for each ε > 0, limu lim supn P[ρ(Xun, Xn) ≥ ε] = 0, then Xn ⇒ X.

通俗解释与例子

定理3.1 的直觉: 如果 Xn 收敛到 X，而 Yn 和 Xn 的距离"依概率趋于零"，那么 Yn 也收敛到 X。这是 Slutsky 引理在度量空间中的推广。

定理3.2 的直觉: 这是"对角线论证"的概率版本。如果我们能用一列"近似" Xn 的随机元来逼近极限，且近似误差可控制，那么 Xn 本身也收敛。这个结果在第7章证明 Donsker 定理的第二种方法中发挥了关键作用。

3.2 积分到极限 (Integration to the Limit) 与 一致可积性

Uniform Integrability / 一致可积性

The Xn are uniformly integrable if lima→∞ supn ∫[|Xn|≥a] |Xn| dP = 0.

中文: 当截断阈值 a → ∞ 时，各 Xn 尾部积分一致趋于零。

Theorem 3.5

If Xn are uniformly integrable and Xn ⇒ X, then X is integrable and E Xn → E X.

中文: 如果 Xn 一致可积且依分布收敛于 X，则 X 可积且期望收敛。

通俗例子

反例（非一致可积导致期望不收敛）: 令 Xn 以概率 1 − 1/n 取 0，以概率 1/n 取 n²。则 Xn ⇒ 0（因为对任何 x > 0，P[Xn > x] → 0），但 E Xn = (1/n)·n² = n → ∞ ≠ 0 = E(0)。这里 Xn 不是一致可积的。

充分条件: 如果 supn E|Xn|1+ε < ∞（存在高阶矩的一致界），则自动一致可积。

§4. Long Cycles and Large Divisors* / 长循环与大素因子*

本章定位

第4节是第1章的应用示范，展示了弱收敛理论（特别是连续映射定理和 ℝ∞ 上的收敛理论）如何用于解决组合数学和数论中的经典问题。这一节用*标记，初次阅读可以跳过而不影响对后续理论的理解。

Original (Cleaned - Core Idea)

Here we study the cycle structure of random permutations on n letters, obtaining in particular the joint limiting distribution (n → ∞) of the greatest cycle lengths. And we derive analogous results on the limiting distribution of the largest prime divisors of an integer drawn at random from among the first n integers. These distributions are described by a certain measure — the Poisson-Dirichlet measure — on a certain space of sequences.

中文翻译

本节研究随机排列的循环结构，特别地给出了最大循环长度的联合极限分布。同时也得到随机整数的最大素因子的类似极限分布。这些分布由一个特定的测度——Poisson-Dirichlet 测度——来描述，定义在一个序列空间上。

核心思想: 随机排列的循环结构

事实1: 在 n 个元素上随机均匀选取一个排列，将排列写成循环的标准形式（第一个循环从1开始，每个新循环从尚未出现的最小元素开始）。则:

  • 第一个循环长度 C₁ 均匀分布在 {1, 2, ..., n} 上: P[C₁ = i] = 1/n
  • 给定 C₁ = i，第二个循环长度 C₂ 均匀分布在 {1, ..., n−i} 上

事实2 (渐近): 令 Lᵢⁿ = Cᵢ / n 为相对循环长度。则 (L₁ⁿ, L₂ⁿ, ...) ⇒ (G₁, G₂, ...)，其中:

G₁ = B₁,   Gᵢ = (1−B₁)(1−B₂)...(1−Bᵢ₋₁)Bᵢ

B₁, B₂, ... 是 i.i.d. Uniform[0,1] 随机变量。

直觉: 这相当于"随机切割单位区间"：先从 [0,1] 随机切下长度为 B₁ 的一段，再从剩余部分切下比例 B₂，依此类推。ΣGᵢ = 1 几乎必然成立。

排序后: 将循环长度从大到小排序（最长循环最受关注），得到 G(1) ≥ G(2) ≥ ...。排序操作在序列空间(A)上是连续的，因此可通过连续映射定理得到极限分布。这就是 Poisson-Dirichlet 分布（参数 θ = 1）。

数值: 最长循环的中位数约为 0.61×n，即随机排列中最长循环的长度超过 0.61n 的概率约为 1/2。

大素因子: 类似地，从 {1,...,n} 中随机选取整数 Nn，其不同素因子按大小排序后取对数，相对大小的极限分布也遵循 Poisson-Dirichlet 分布！

§5. Prohorov's Theorem / Prohorov定理

核心定理 — 全书最重要的理论支柱

Prohorov 定理是弱收敛理论中最重要的结构定理。它将两个看似无关的概念——相对紧性（relative compactness）和紧性（tightness）——联系起来，为证明概率测度序列的弱收敛提供了强大工具。

定义: 相对紧性 (Relative Compactness)

A family Π of probability measures on (S, S) is relatively compact if every sequence of elements of Π contains a weakly convergent subsequence; that is, if for every sequence {Pn} in Π there exist a subsequence {Pnᵢ} and a probability measure Q such that Pnᵢ ⇒ Q.

中文: 相对紧的测度族是指其中任何序列都有弱收敛子列（且子列的极限必须是概率测度，即总质量为1）。

定义: Tightness / 紧性（测度族版本）

The family Π is tight if for every ε > 0 there exists a compact set K such that PK > 1 − ε for every P in Π.

中文: 对每个 ε > 0，存在一个紧集 K，使得族中所有测度都在 K 上有质量至少 1−ε。

Theorem 5.1 (Prohorov's Theorem — Direct Half)

If Π is tight, then it is relatively compact.

中文: 紧 ⇒ 相对紧。

Theorem 5.2 (Prohorov's Theorem — Converse Half)

Suppose that S is separable and complete. If Π is relatively compact, then it is tight.

中文: 在 Polish 空间上，相对紧 ⇒ 紧。

为什么需要 Prohorov 定理？

典型应用模式（例5.1）: 在 C[0,1] 上，要证明 Pn ⇒ P，我们需要做两件事：

  1. 证明 {Pn} 是 tight 的（这通常通过验证路径的"等度连续"条件来完成）
  2. 证明有限维分布收敛（通常通过中心极限定理可得）

然后由 Prohorov 定理，{Pn} 是相对紧的，任何子列都有收敛子子列。由于有限维分布收敛到 P 的有限维分布，而有限维集合是分离类（唯一性），所有子子列的极限必须是 P。由定理2.6，整个序列 Pn ⇒ P。

这个模式在第2章和第3章被反复使用!

反例: 什么情况下 tightness 失败？

实数线上的例子:

  1. 质量逃逸到无穷: Pn = δn。对任何紧集 K（有界），当 n 大时 PnK = 0。
  2. 质量"蒸发": 令 Pn 是 [-n, n] 上的均匀分布与 δ₀ 的混合（各半）。任何子列的"非 δ₀"部分的质量在极限中消失，不能收敛到概率测度。

C[0,1] 上的反例（例5.4）: Pn = δzn，其中 zn 是第1.5节中的"帐篷函数"。zn 逐点趋于0但不一致收敛。序列 {Pn} 不是 tight 的，因为任何紧集 K（由 Arzelà-Ascoli，K 中的函数必须等度连续）不能包含所有的 zn。

Prohorov定理证明要点 (Theorem 5.1)

证明是构造性的，使用了精细的"对角线论证"(diagonal argument):

  1. 选取一列紧集 Ku 使得 PnKu > 1 − u−1
  2. 构造一列可数个开集 A，使得每个开集可由 A 中的集合逼近
  3. 通过取子列确保所有 H ∈ H（由 A 中集合生成的有限并类）上的极限 α(H) = lim PnᵢH 存在
  4. 从 α 构造外测度 γ，证明 γ 限制到 Borel 集上即为所求的概率测度 P

这个证明是 Caratheodory 外测度构造法的经典应用。

Theorem 6.7 — Skorohod Representation Theorem

Suppose that Pn ⇒ P and P has a separable support. Then there exist random elements Xn and X, defined on a common probability space, such that L(Xn) = Pn, L(X) = P, and Xn(ω) → X(ω) for every ω.

中文: 如果弱收敛 Pn ⇒ P 且 P 有可分支撑，则存在定义在同一个概率空间上的随机元 Xn, X，使得它们各自的分布分别为 Pn 和 P，且 Xn(ω) → X(ω) 处处成立（不仅仅是依概率或几乎处处）。

Skorohod 定理的意义

"几乎是魔术": 弱收敛本身只说积分收敛（"弱"），但 Skorohod 定理告诉我们：只要极限测度有可分支撑，我们总能找到适当的构造使得逐点收敛成立。这意味着许多仅需逐点收敛的论证可以直接应用于弱收敛的情形。

应用举例: 连续映射定理可以通过 Skorohod 定理得到另一种简单的证明：构造逐点收敛的 Xn → X，则 h(Xn) → h(X) 逐点成立，从而依分布收敛。

§7. Weak Convergence and Tightness in C / C中的弱收敛与紧性

Theorem 7.1 — 基本策略

Let Pn, P be probability measures on (C, C). If the finite-dimensional distributions of Pn converge weakly to those of P, and if {Pn} is tight, then Pn ⇒ P.

中文翻译与解释

如果 {Pn} 满足：

  1. 有限维分布收敛: 对任意 k 和 t₁,...,tₖ，随机向量 (Xt₁,...,Xtₖ) 的分布在 Pn 下弱收敛到在 P 下的分布
  2. {Pn} 是 tight 的

则 Pn ⇒ P。这是将 Prohorov 定理应用于 C 空间的直接推论（结合有限维集合是分离类这一事实）。

7.1 Arzelà-Ascoli 特征 & Tightness 判定

Theorem 7.2 — Arzelà-Ascoli

The set A ⊂ C is relatively compact iff:
(7.3) supx∈A |x(0)| < ∞ (bounded at 0)
(7.4) limδ→0 supx∈A wx(δ) = 0 (uniform equicontinuity)

中文: A 相对紧当且仅当 (i) 在0点一致有界；(ii) 函数族一致等度连续（uniformly equicontinuous）。

Theorem 7.3 — Tightness 的实用判据

{Pn} is tight iff:

(i) lima→∞ lim supn Pn[x: |x(0)| ≥ a] = 0

(ii) limδ→0 lim supn Pn[x: wx(δ) ≥ ε] = 0 for each ε > 0

中文: (i) 在0点的值不逃逸到无穷；(ii) 连续性模在概率意义下一致趋于零。

直观理解 tightness 在 C 中的含义

条件(i) 防止函数在起点处"爆掉"。

条件(ii) 才是核心——它要求：对任意小的 δ，函数在所有 n 上的振荡（|x(s)−x(t)|, |s−t|≤δ）一致地小。这本质上等价于说：样本路径"不会太振荡"。

为什么帐篷函数序列不是 tight? 因为 zn(t) 在长度 1/n 的区间上从 0 上升到 1 再降回 0，所以 w(zn, δ) = 1 对所有 n ≥ 1/δ 成立 → 条件(ii) 失败。

Theorem 7.4 & Corollary — 使用分划的简化

If 0 = t₀ < t₁ < ... < tv = 1 and min(tᵢ − tᵢ₋₁) ≥ δ, then wx(δ) ≤ 3·maxi sups∈[tᵢ₋₁,tᵢ] |x(s) − x(tᵢ₋₁)|.

This gives a practical way to bound the modulus wx(δ) using a grid — key to proving tightness in Donsker's theorem.

§8. Wiener Measure and Donsker's Theorem / Wiener测度与Donsker定理

历史背景

Wiener 测度是 C[0,1] 上最重要的概率测度——它为 Brownian 运动（Brownian motion） 提供了严格的数学模型。Norbert Wiener 在1923年首次构造了这个测度。Donsker 在1951年证明的"不变原理"将随机游动与 Brownian 运动联系起来，是概率论中最重要的极限定理之一。

8.1 Wiener 测度的定义与构造

Wiener 测度 W 的两个定义性质

(i) 在 W 下，每个坐标变量 xt 服从均值为0、方差为 t 的正态分布: W[xt ∈ H] = (2πt)−1/2 ∫H e−u²/2t du

(ii) 在 W 下，增量是独立的: 若 0 ≤ t₀ ≤ t₁ ≤ ... ≤ tk ≤ 1，则 xt₁−xt₀, ..., xtₖ−xtₖ₋₁ 相互独立。

由此推出: 对 s ≤ t，xt − xs ~ Normal(0, t−s)；且 Cov(xs, xt) = s ∧ t。

构造思路: 从随机游动到Brownian运动

令 ξ₁, ξ₂, ... 为 i.i.d. 随机变量，Eξᵢ = 0，Var(ξᵢ) = σ² > 0。令 Sₖ = ξ₁ + ... + ξₖ。

定义随机函数 Xn ∈ C：在点 i/n 处取值为 Sᵢ/(σ√n)，在之间线性插值。

则 Xn 的有限维分布收敛到 Wiener 测度指定的分布（由 Lindeberg-Lévy CLT）。

关键步骤: 证明 {Pn}（Xn 的分布）是 tight 的。这需要验证定理7.3中的条件(ii) — 路径不要太振荡。

证明 tightness 用到 Etemadi 不等式（独立情形下的极大不等式）和 CLT。一旦 tightness 成立，Prohorov 定理给出弱收敛子列，且该子序列的有限维分布收敛到目标，因此子序列极限就是 Wiener 测度。

具体例子: 对称简单随机游动 → Brownian运动

设置: 抛一枚公平硬币 n 次。每次正面朝上 +1，反面 −1。令 Sk 为前 k 次抛掷的部分和。通过线性插值构造连续路径 Xn(t) = S⌊nt⌋/√n。

尺度变换的含义:

  • 空间缩放 (÷√n): 由 CLT 可知 Sn/√n ~ Normal(0,1)，所以要除 √n 才能得到非退化的极限
  • 时间缩放 (×n): 将 n 步压缩到单位区间 [0,1] 上，使得每个时间点对应约 nt 步

结果: 当 n → ∞ 时，随机游动路径（缩放后）在 C[0,1] 中弱收敛到 Brownian 运动。

这就是 Donsker 定理!

8.2 Donsker定理与不变原理

Theorem 8.2 — Donsker's Theorem (Functional CLT)

If ξ₁, ξ₂, ... are i.i.d. with mean 0 and variance σ² > 0, and if Xn is the random function defined by linear interpolation from Sk/(σ√n) at k/n, then Xn ⇒ W.

为什么叫"不变原理" (Invariance Principle)?

极限 W（Brownian 运动）不依赖于 ξᵢ 的具体分布——只要均值为0、方差有限即可。无论 ξᵢ 是正态、二值(±1)、均匀还是任何其他满足条件的分布，部分和过程在缩放后都收敛到同一个 Brownian 运动。这就是"不变"的含义。

Donsker定理的应用: 最大值分布

令 Mn = max0≤i≤n Si 为部分和的最大值。由 Donsker 定理和连续映射定理：

Mn/(σ√n) = supt Xn(t) ⇒ supt W(t)

利用简单随机游动的反射原理可以算出 supt W(t) 的分布：

P[supt W(t) ≤ a] = (2/π)½ ∫0a e−u²/2 du  (a ≥ 0)

因此对任意满足 Donsker 定理条件的 ξᵢ，Mn/√n 的极限分布都是这个"折叠正态分布"(folded normal)。这就是不变原理的强大之处。

反射原理 (Reflection Principle) 简述

对于对称简单随机游动 (ξᵢ = ±1，各概率 ½)，有以下精确公式：

P[Mn ≥ a, Sn = v] = P[Sn = 2a − v]  (v ≤ a, a > 0)

直觉: 将首次达到高度 a 之后的路径关于水平线 y = a 做镜面反射。反射后的每一条"不良"路径恰好对应一条以 2a−v 结束的路径。因此，达到 a 且以 v 结束的路径数 = 以 2a−v 结束的路径数。

8.3 Brownian 桥 (The Brownian Bridge)

Brownian Bridge W°

W° 是中心化的 Gaussian 随机函数，满足 E[W°t] = 0 和 E[W°s W°t] = s(1−t) for s ≤ t。

最简单构造: W°t = Wt − t·W1。

直观: Brownian 桥是"条件化"在 W1 = 0 的 Brownian 运动（虽然这是零概率事件，但可通过取极限严格定义）。

§9. Functions of Brownian Motion Paths / Brownian运动路径的泛函

第9节核心思想

一旦 Donsker 定理给出了 Xn ⇒ W，我们可以通过连续映射定理得到各种路径泛函的极限分布。许多泛函在简单随机游动情形下可以通过组合方法精确计算，然后取极限得到 Brownian 运动对应泛函的分布。

最大值与最小值联合 无穷级数表示（公式9.10） 涉及无穷多个正态概率

§10. Maximal Inequalities / 极大不等式

为什么需要极大不等式

第8节用 Etemadi 不等式证明了 tightness，但该不等式依赖独立性假设。对于相依序列（如三角级数的部分和、数论中的函数），需要不依赖独立性的方法。第10节发展了一套基于 mijk（三点最小值）的极大不等式体系。

核心不等式 (Theorem 10.1)

定义 mijk = min{|Sj−Si|, |Sk−Sj|} 和 Ln = max0≤i≤j≤k≤n mijk。

如果 P[mijk ≥ λ] ≤ (∑i<l≤k ul)2α / λ4β (α>½, β≥0)，则: P[Ln ≥ λ] ≤ K(∑l≤n ul)2α / λ4β

关键洞察

Ln 控制了部分和序列的"振荡程度"。mijk 衡量从 i 到 j 和从 j 到 k 的两段中较小者的长度。如果 Ln 小，则部分和在小的区间上变化不大——这正是 tightness 需要的。

定理10.1的优雅之处在于：只需对所有的三点组 (i,j,k) 验证一个简单的尾概率界，就能得到 Ln 的界。这不需要独立性！

§12. The Geometry of D / D空间的几何

Original (Cleaned)

Let D = D[0,1] be the space of real functions x on [0,1] that are right-continuous and have left-hand limits:

(i) For 0 ≤ t < 1, x(t+) = lims↓t x(s) exists and x(t+) = x(t).

中文翻译

D[0,1] 是定义在 [0,1] 上满足以下条件的实值函数空间：

(ii) 左极限存在（left-hand limits exist）: 对 0 < t ≤ 1，左极限 x(t−) 存在

这类函数称为 càdlàg 函数（法语"右连续、左极限"的缩写）。

12.1 Skorohod 拓扑 — 为什么需要它？

核心动机: 为什么C空间不够？

Poisson 过程的例子: 考虑速率为 n 的 Poisson 过程 Nn(t)，中心化并标准化为 Xn(t) = (Nn(t) − nt)/√n。Xn 有跳跃（在 Poisson 事件时刻），所以它不是连续的。但我们知道 Xn 应该收敛到 Brownian 运动 W。

问题：如果用一致拓扑（C 中的度量），Xn 根本不收敛到 W！因为 Xn 有跳跃而 W 没有。但如果我们允许时间轴上有小的扰动——即允许函数值在稍微不同的时间点上比较——那么 Xn 的跳跃可以被"吸收"到连续极限中。

这就是 Skorohod 拓扑的核心思想：同时允许时间变形和空间变形。

Skorohod 度量 d(x,y)

d(x,y) = inf{ε > 0 : ∃ λ ∈ Λ such that ||λ − I|| < ε and ||x − y∘λ|| < ε}

其中 Λ = {严格递增的连续双射 λ: [0,1]→[0,1], λ(0)=0, λ(1)=1}。

中文: 存在一个"时间变形" λ（接近恒等映射），使得变形后的 y 在一致意义下接近 x。

具体例子: 跳跃如何"收敛"到连续函数

令 zn(t) = I[0.5, 0.5+1/n)(t)（在 0.5 处有高度为1、宽度为 1/n 的脉冲），和 z(t) ≡ 0。

一致拓扑下: ||zn − z|| = 1，不收敛。

Skorohod 拓扑下: 取 λn 使得 λn(0.5) = 0.5 + 1/(2n)，且在附近做小调整。则 zn∘λn ≡ 0（因为脉冲被移到了 [0.5+1/(2n), 0.5+3/(2n)) 而 zn 在此区间外为0），且 ||λn − I|| → 0。故 d(zn, z) → 0。

关键洞察: Skorohod 拓扑"容忍"小的水平位移。一个窄脉冲可以通过微调时间轴被"抹除"。这恰好是概率极限理论需要的：Poisson 过程的微小跳跃在 Skorohod 拓扑下可以被收敛到 Brownian 运动的连续路径。

12.2 d₀ 度量、完备性与可分性

d₀ 度量 (Complete Metric)

d₀(x,y) = inf{ε > 0 : ∃ λ ∈ Λ such that ||λ||° < ε and ||x − y∘λ|| < ε}

其中 ||λ||° = sups≠t |log((λ(t)−λ(s))/(t−s))| 控制 λ 的斜率偏离1的程度。

d vs d₀ 的区别

d 只要求 ||λ − I|| 小（时间变形在 sup 范数下小）。d₀ 进一步要求 λ 的斜率处处接近1（在对数尺度下）。

d₀ 更强（d₀ 收敛 ⇒ d 收敛），但两者诱导相同的拓扑。然而 d₀ 是完备的而 d 不是。D[0,1] 在 d₀ 下是 Polish 空间（完备+可分），这对应用 Prohorov 定理是必要的。

D中的连续性模 w′ (Modulus for D)

w′(x, δ) = inf{tᵢ} δ-sparse maxi wx[ti−1, ti)

取infimum在所有δ-稀疏的分划{tᵢ}（即 ti − ti−1 > δ）上。

w′ 与 w 的关系

对于连续函数: w′(x, δ) = w(x, δ)（两者一致）。这解释了为什么在 C 中子空间上 Skorohod 拓扑退化为一致拓扑。

对于有跳跃的函数: w(x, δ) 不趋近于0（因为在跳跃点附近振荡很大），但 w′(x, δ) → 0。w′ 通过允许在"大跳跃"处分段，凝练出"跳跃之间的连续部分"的振荡信息。

§13. Weak Convergence and Tightness in D / D中的弱收敛与紧性

完全平行于C的理论

D 中的弱收敛与紧性理论是 C 中对应理论的直接推广——只需用 w′ 替换 w，并适当处理跳跃行为。

Theorem 13.2 — Tightness Criterion in D

{Pn} is tight in D iff:

(i) lima→∞ lim supn Pn[x: ||x|| ≥ a] = 0

(ii) limδ→0 lim supn Pn[x: w′(x, δ) ≥ ε] = 0 for each ε > 0

另外还需要控制跳跃: limδ→0 lim supn Pn[x: j(x) ≥ ε] = 0（最大跳跃不超过 ε），但在很多情况下 j(x) 本身由 w′ 控制。

D中 tightness 的应用: 经验过程

考虑 i.i.d. 样本 U1,...,Un ~ Uniform[0,1]。经验分布函数 Fn(t) = (1/n)∑ I[Ui ≤ t] 是 D[0,1] 中的随机元（右连续、有跳跃）。标准化经验过程 √n(Fn(t) − t) 在 D 中弱收敛到 Brownian 桥 W°。

这个结果（Donsker 定理的经验过程版本，不要与第8节的 Donsker 定理混淆！）在非参数统计中具有基础重要性。它给出了 Kolmogorov-Smirnov 统计量等拟合优度检验的渐近分布。

§17. More on Prime Divisors / 素数因子的进一步讨论

17.1 引言与一般极限定理

Original (Cleaned)

Section 4 treats, among other things, the asymptotic distribution of the largest prime divisors of a random integer. Here we study the total number of prime divisors of a random integer and the number of them in a given range, and we show that the limit is normal in some cases and Poisson in others. And we prove the associated functional limit theorems.

On the σ-field of all subsets of ℕ = {1,2,...}, let Pn be the probability measure corresponding to a mass of 1/n at m for 1 ≤ m ≤ n. Let δp(m) be 1 or 0 according as p divides m or not. By the fundamental theorem of arithmetic, the events "divisible by p" and "divisible by q" are approximately independent for distinct primes.

Define fn(m) = Σp≤n fn(p)δp(m). Consider independent random variables ξp, one for each prime p, that take values 1 and 0 with probabilities p−1 and 1 − p−1. Define μn = Σp≤n fn(p)/p and σ2n = Σp≤n f2n(p)(1/p − 1/p2). Assume σ2n > 0.

中文翻译

第4节处理了随机整数的最大素因子的渐近分布。本节我们研究随机整数中全体素因子的数量以及在给定范围内素因子的数量，并证明在某些情况下极限是正态的，在其他情况下是Poisson的。我们还证明相关的泛函极限定理。

在 ℕ = {1,2,...} 的所有子集构成的σ-域上，令 Pn 为在 1 ≤ m ≤ n 处赋予质量 1/n 的概率测度。令 δp(m) 为1或0，取决于 p 是否整除 m。由算术基本定理，对于不同的素数 p 和 q，事件"被 p 整除"和"被 q 整除"近似独立。

定义 fn(m) = Σp≤n fn(p)δp(m)。考虑独立随机变量 ξp，每个素数 p 对应一个，以概率 p−1 和 1 − p−1 取值1和0。定义 μn 和 σ2n 如上。假设 σ2n > 0。

Theorem 17.1 (一般极限定理)

假设存在正的常数 an 使得:

Σ |fn(p)| → 0,    an/nε → 0

对每个正 ε 成立。如果 (Sn − μn)/σn ⇒ ξ 且 ξ 的分布由其矩决定，则 (fn − μn)/σn ⇒ ξ。

直观: 素因子之间的依赖关系非常弱（几乎是独立的），只要控制"大素数"对三角阵列的影响（通过截断参数 an），算术和就表现得像真正的独立变量和。

Example 17.1 (Erdos-Kac 定理)

取 fn(p) ≡ 1，则 fn(m) 是 m 的全体素因子的总数。由数论事实 Σp≤n 1/p = log log n + O(1)，可得 μn 和 σ2n 均为 log log n + O(1)。取 an = n1/log log n 满足定理17.1的条件，得:

Pn[m: ν(m) ≤ x] ⇒ Φ(x)

其中 ν(m) = Σp|m 1 适当标准化。除以 log log n 得到Hardy-Ramanujan弱大数律: ν(m)/log log n ⇒ 1。

通俗解释: 一个大整数的不同素因子数量，近似服从正态分布，其均值和方差都约为 log log n。这个结果非常惊人——数论中完全确定性的对象居然显示出如此"随机"的行为。

Example 17.2 (泛函版本)

若 |fn(p)| < ∞ 且 σn → ∞，取 an = n1/σ2n，则可证明:

(fn − μn)/σn = σ−1n Σp<n fn(p)(δp − p−1) ⇒ N

Example 17.3 (Poisson极限情形)

设 fn(p) = 1 当且仅当 un < p ≤ vn，即 fn(m) 是 m 在 (un, vn] 内素因子的个数。若 vn = o(nε) 且 un → ∞（对每个 ε > 0），则 fn ⇒ ξ(λ)，其中 ξ(λ) 具有均值为 λ 的Poisson分布，且 λ = log t − log s（当取 un = escn，vn = etcn）。

通俗解释: 如果只数那些在特定范围内（既不太小也不太大的素数）的素因子，它们的数量服从Poisson分布。这类似于"稀有事件"的渐近分布，与§4中的Poisson-Dirichlet极限形成对比。

Original (Cleaned)

Znt(m) = (log log m)−1/2 Σ(δp(m) − p−1)

Theorem 17.3 (Poisson过程极限)

对每个 m 的素因子 p，在 log log p − log log log n 处放置一个点。令 Xnt(m) 为 (0,t] 内的点数（t ≥ 0）或负的 (t,0] 内的点数（t ≤ 0）。则 Xn ⇒ X（在 D(−∞, ∞) 意义下），其中 X 是强度为1的标准Poisson过程。

§18. Martingales / 鞅

章节概览

核心思想: 鞅差序列(martingale differences)是介于独立变量和一般相依变量之间的自然中间概念。由于鞅差在给定过去的条件下均值为0，许多独立变量的极限理论可以推广到鞅差阵列。

应用动机: 平稳Markov链的对数似然函数的导数是鞅差序列，其大样本理论依赖于鞅的中心极限定理。

18.1 三角阵列

Definition (鞅差三角阵列)

对于每个 n，ξn1, ξn2, ... 是关于σ-域 Fn1 ⊂ Fn2 ⊂ ... 的鞅差: ξnk 是 Fnk-可测的，且 E[ξnk | Fn,k−1] = 0。设 σ2nk = E[ξ2nk | Fn,k−1]。

Theorem 18.1 (鞅差CLT)

假设:

(18.1)    Σk=1∞ σ2nk →P σ2

(18.2)    Σk E[ξ2nk · 1{|ξnk|>ε}] → 0  

对每个正 ε 成立。则 Σk ξnk ⇒ σN。

条件解读: (18.1)是"条件方差收敛"，(18.2)是Lindeberg条件——控制大跳跃对标准化和的贡献。

Theorem 18.2 (鞅差泛函CLT)

在Theorem 18.1的设定下，定义 Xnt = Σk≤nt ξnk。假设:

(18.3)    Σk≤nt σ2nk →P σ2t

对每个 t 成立，且(18.2)对每个 t 和 ε 成立。则 Xn ⇒ W（在 D[0,∞) 的Skorohod拓扑下）。

18.2 平稳遍历鞅差

Theorem 18.3 (平稳遍历鞅差的泛函极限定理)

设 ..., ξ−1, ξ0, ξ1, ... 是平稳、遍历的鞅差序列（ξn 是 Fn-可测且 E[ξn|Fn−1] = 0，其中 Fn = σ[ξk: k ≤ n]）。若 E[ξ21] = σ2 > 0 且有限，定义:

Xnt = (σ√n)−1 Σk≤nt ξk

则 Xn ⇒ W（在 D[0,∞) 意义下）。

证明思路: 在序列空间上表示，利用平稳性和遍历定理验证条件(18.3)；条件(18.2)由 E[ξ21] < ∞ 保证。

Example 18.1 (平稳Markov链的统计应用)

设 (ξ0, ξ1, ...) 是有限状态空间的平稳Markov链，转移概率和稳态概率 p(i,j;θ) 和 p(i;θ) 均为正且依赖参数 θ。对数似然为 Ln(θ) = log p(ξ0;θ) + Σk=1n log p(ξk−1, ξk;θ)。

导数 L'n(θ) 的加项构成平稳遍历鞅差。由Theorem 18.3:

n−1/2L'n(θ) ⇒ σ(θ)N

其中 σ2(θ) = E[ξ21]。这是Markov链大样本推断的理论基础。

§19. Ergodic Processes / 遍历过程

章节概览与核心方法

基本问题: 对于一个一般的平稳遍历过程，如何建立CLT和泛函CLT？

Gordin方法: 将原始平稳过程用鞅差来逼近，然后应用§18的鞅理论。关键条件(19.3)确保"信息随时间衰减"：Σn=1∞ ||E[ξn|F0]|| < ∞。

19.1 基本定理

Theorem 19.1 (平稳遍历过程的CLT)

假设 (ξn)n∈ℤ 是平稳的、遍历的过程，满足:

(19.3)    Σn=1∞ ||E[ξn|F0]|| < ∞

且 E[ξ2n] < ∞。则 E[ξ1] = 0（否则减去均值），且:

σ2 = E[ξ21] + 2 Σk=1∞ E[ξ1ξ1+k]

绝对收敛。如果 σ > 0，则 Sn/σ√n ⇒ N；泛函版本 Xn ⇒ W（在 D[0,∞) 意义下）。

Original (Cleaned) - Gordin's Method

The idea behind Gordin's method: define ηk = ξk + θk − θk−1 where θk = Σi=1∞ E[ξk+i|Fk]. Because of (19.12) (which follows from (19.3)), the series converges. Then E[ηk|Fk−1] = 0, so ηn is a martingale difference. But Mn = Σk=1n ηk = Sn + θn − θ0, and since ||θn|| is finite, n−1/2(Mn − Sn) → 0.

中文翻译 - Gordin方法详解

Gordin方法的核心思想: 定义 ηk = ξk + θk − θk−1，其中 θk = Σi=1∞ E[ξk+i|Fk]。由于(19.3)，此级数收敛。可以验证 E[ηk|Fk−1] = 0，即 ηn 是鞅差。

但 Mn = Σk=1n ηk = Sn + θn − θ0。由于 ||θn|| 有限（平稳性），n−1/2(Mn − Sn) → 0。因此 Sn 的渐近行为由 Mn 的鞅极限理论决定。

直观理解: 这个方法是对m-相依情形的无穷推广。对于1-相依过程，校正项只有一项: ηk = ξk + E[ξk+1|Fk] − E[ξk|Fk−1]。对于一般的平稳过程，只要条件期望以足够快的速度衰减（19.3），无穷级数的校正仍然有效。

Example 19.1 (m-相依过程)

若 (ξn) 是 m-相依的（距离超过 m 的变量独立），则 E[ξn|F0] = 0 对于 n > m 成立，(19.3)自动满足。

Example 19.2 (自回归过程)

定义 ξn = Σi=0∞ ρiεn−i，其中 |ρ| < 1 且 εn 是i.i.d.，均值为0方差为1。则:

||E[ξn|F0]|| = ρ2n/(1 − ρ2)

(19.3)成立，因为 Σ ρn < ∞。且该过程是遍历的。Theorem 19.1适用。

19.2 均匀混合 (Uniform Mixing)

Definition (三种混合系数)

对于平稳过程 (ξn)，定义 Fk = σ[..., ξk−1, ξk]，Gk+n = σ[ξk+n, ξk+n+1, ...]:

系数定义含义
αnsup{|P(A∩B) − PA·PB|: A∈Fk, B∈Gk+n}α-混合: 最弱条件
ρnsup{|E[ξη]|: ξ∈Fk, η∈Gk+n, Eξ=Eη=0, ||ξ||≤1, ||η||≤1}ρ-混合: 基于相关性
φnsup{|P(B|A) − PB|: A∈Fk, PA>0, B∈Gk+n}φ-混合: 最强条件

有关系 αn ≤ ρn ≤ 2√αn。

Theorem 19.2 (ρ-混合过程的CLT)

假设 Σn ρn < ∞，且 E[ξ21] < ∞。则Theorem 19.1的结论成立。特别地，Σ ρn < ∞ 保证了遍历性以及条件(19.3)。

Example 19.3 (不可约有限状态Markov链)

对于具有有限状态空间的可约、非周期Markov链，存在常数 K 和 θ ∈ (0,1) 使得 |p(n)uv/pv − 1| < Kθn。由此 φn ≤ Kθn 且 Σ ρn < ∞ 自动满足。这是Markov链Monte Carlo的理论基础之一。

19.3 混合过程的泛函

Theorem 19.3 (混合过程的泛函CLT)

假设 Σn ρn < ∞。设 ηn = f(..., ξn−1, ξn, ξn+1, ...) 具有零均值和有限方差。若存在函数 ηkn = fk(ξn, ..., ξn+k) 使得:

Σk ||η1 − ηk1|| < ∞

则 (ηn) 也满足Theorem 19.1的结论。这意味着如果 (ξn) 是混合的，那么适当的Lipschitz型泛函 f 生成的序列仍然满足CLT。

Example 19.4 (线性过程)

设 ξn 是均值为0方差为1的i.i.d.变量，定义:

ηn = Σi=0∞ αiξn−i

其中 Σ α2i < ∞。取 ηkn = Σi=0k αiξn−i，则 ||η1 − ηk1||2 = Σi=k+1∞ α2i。若该尾和以更快速度衰减，则 Σ ||η1 − ηk1|| < ∞ 满足，Theorem 19.3适用。

Example 19.5 (单位区间上的二元展开)

令 P 为Lebesgue测度，ξn(x) 为 x 的二元展开的第 n 位数字。这是平稳独立的过程，因而是遍历的。对于 ηn(x) = f(Tnx)（其中 T 是 Tx = 2x mod 1），若 f 连续且满足Holder条件 |f(x)−f(y)| ≤ K|x−y|β（β > 0），则(19.29)成立且存在泛函CLT。

通俗理解: 只需用根据前 k 位数字定义的函数逼近原始函数 f，且逼近误差以足够快的速度衰减，就能保证CLT成立。

Chapter 5: Other Modes of Convergence
第五章: 其他收敛模式

章节概览 / Chapter Overview

核心问题: Donsker定理是依分布收敛(1")的结果。能否加强到依概率收敛(2")甚至几乎必然收敛(3")? 第5章给出三种不同的加强方式:

  • §20 — 依概率收敛: 利用Skorohod嵌入定理，在Brownian运动路径上构造部分和的精确表示（不是仅依分布等价，而是轨道级别的逼近）
  • §21 — 独立正态序列逼近: 在原始概率空间（或扩张空间）上构造一个独立标准正态序列，使得从某个指标开始，部分和之差以概率被控制
  • §22 — Strassen定理（重对数律）: 用 √(2n log log n) 而非 √n 标准化，证明随机函数序列的极限点集合恰好是K（绝对连续函数加L2约束）

§20. Convergence in Probability / 依概率收敛

Original (Cleaned)

There are four major modes of convergence for random variables: 1° convergence in distribution, 2° convergence in probability, 3° convergence with probability 1, 4° convergence in L2. Neither of 3° and 4° implies the other, but each implies 2°, and 2° implies 1°.

Donsker's theorem and most of the other results of the preceding chapters concern convergence in distribution. Instances of convergence in distribution, even for random variables, often cannot be strengthened as they stand to convergence in probability. But sometimes, if ηn ⇒ ξ cannot be replaced by convergence in probability, it is possible to prove (on a new probability space) that ηn − ξn → 0, where each ξn has the distribution of ξ. In this section and in the next, Donsker's theorem will be strengthened in this way.

中文翻译

随机变量有四种主要收敛模式: 1° 依分布收敛，2° 依概率收敛，3° 以概率1收敛（几乎必然收敛），4° L2收敛。3° 和 4° 互不蕴含，但各自蕴含 2°，而 2° 蕴含 1°。

Donsker定理和前面章节的大多数结果涉及依分布收敛。即使是随机变量的依分布收敛，通常也不能直接加强为依概率收敛。但有时，虽然 ηn ⇒ ξ 不能替换为依概率收敛，却可以（在一个新的概率空间上）证明 ηn − ξn → 0，其中每个 ξn 与 ξ 同分布。本节和下一节将以这种方式加强Donsker定理。

Theorem 20.1 (Skorohod嵌入定理)

在某个概率空间 (Ω', F', P') 上存在标准Brownian运动 {B(t): 0 ≤ t < ∞} 以及一个非降的随机变量序列 τ0 = 0, τ1, τ2, ...，使得:

  • 增量 ξn = B(τn) − B(τn−1) 是独立的，且与原始和项有相同分布
  • 等待时间差 τn − τn−1 是独立同分布的，均值为1（如果原始变量有方差 σ2，则均值为 σ2）
Theorem 20.2 (依概率收敛版本的Donsker定理)

设 ξ1, ξ2, ... 是i.i.d.，均值为0方差为1。在定理20.1的构造下，令 Tk = Σi≤k ξi = B(τk)，定义 Yn 为线性插值 Yni/n = Ti/√n。又定义 Bn(t) = B(nt)/√n（与 W 同分布）。则:

||Yn − Bn|| →P' 0

其中 ||·|| 是上确界范数。这蕴含着Donsker定理（L(Yn) = L(Xn)，且 L(Bn) = L(W)），但给出了更强的结论: 整个随机函数的轨道在概率下是接近的。

证明思路

证明分两步: (20.3) ||Bn − Zn|| → 0 和 (20.4) ||Yn − Zn|| → 0，其中 Zn 是线性插值 Zni/n = Bn(i/n)。

(20.3)利用Brownian运动的连续性模估计: P'[sups≤t|B(s)| ≥ a] ≤ 4e−a2/2t。

(20.4)利用强大数律 τi/i → 1 和Brownian运动的连续路径性质.

§21. Approximation by Independent Normal Sequences / 独立正态序列逼近

Original (Cleaned)

The approximation theorem is this: Let ξ1, ξ2, ... be i.i.d. with mean 0 and variance 1. Suppose there is a random variable U independent of {ξi} and uniformly distributed over [0,1]. Then there exists on (Ω, F, P) a sequence η1, η2, ... of standard normal random variables such that:

(21.1)    P[maxk≤n |Sk − Tk| ≥ ε√n] ≤ Kε,    n > n0(ε)

where Sk = Σi≤k ξi, Tk = Σi≤k ηi, and K is an absolute constant. This implies Donsker's theorem and is sometimes called the KMT approximation or "Hungarian construction."

中文翻译

逼近定理(T.21.1): 设 ξi 是均值为0方差为1的i.i.d.序列。假设存在与 {ξi} 独立的 U ~ Uniform[0,1]。则存在标准正态的独立序列 η1, η2, ... 使得:

(21.1)    P[maxk≤n |Sk − Tk| ≥ ε√n] ≤ Kε,    n > n0(ε)

其中 Sk = Σi≤k ξi，Tk = Σi≤k ηi，K 是绝对常数。这蕴含着Donsker定理，但更强: 随机游走的部分和可以被独立正态变量的部分和以高概率在 o(√n) 精度内逼近。有时称为KMT逼近或"匈牙利构造"。

Borel同构与关键引理

证明需要使用Borel同构的概念。关键事实: ℝ ~ (0,1) ~ [0,1] ~ [0,1]∞ ~ ℝ∞（所有这些空间都是Borel同构的），因为每个不可数、可分、完备的度量空间都与 [0,1] Borel同构。

Lemma 1: 如果 L(σ) = μ 且 u ~ Uniform[0,1] 与 σ 独立，则存在随机元 τ（σ 和 u 的函数）使得 L(σ, τ) = ν，其中 ν 是给定边际分布的联合分布。

Lemma 2: 推广到三个空间，保持独立性。

两种证明: 第一种利用Theorem 20.2和Lemma 1直接构造；第二种（Philipp提供）避免Skorohod表示定理，而是通过Prohorov距离和分块构造来逼近，虽然更复杂但可推广到向量值过程。

§22. Strassen's Theorem / Strassen定理

Original (Cleaned)

Let ξ1, ξ2, ... be i.i.d. with mean 0 and variance 1. Let Sn = Σi≤n ξi, and let Xn be the random element of C = C[0,1] defined by:

(22.1)    Xnt = S[nt]/√(2n log log n)

Strassen's theorem has to do not with the asymptotic distribution of Xn, but with the properties of the entire sequence (X3(ω), X4(ω), ...) for each individual ω in a set of probability 1.

Let K be the set of absolutely continuous functions x in C for which x(0) = 0 and the associated x' satisfies ∫01 (x'(t))2 dt ≤ 1.

中文翻译

设 ξ1, ξ2, ... 是均值为0方差为1的i.i.d.序列。定义:

(22.1)    Xnt = S[nt]/√(2n log log n)

与Donsker定理不同，Strassen定理不关心 Xn 的渐近分布（对于固定大的 n，ω 遍及 Ω），而是关心整个序列 (X3(ω), X4(ω), ...) 在几乎每个 ω 处的性质。

令 K 为 C 中满足 x(0) = 0 的绝对连续函数 x 的集合，且 ∫01 (x'(t))2 dt ≤ 1。注: K 是 L2 中的单位球在 C 中的嵌入，是非稠密(nonwhere dense)的紧凸集。

Theorem 22.1 (Strassen定理)

以概率1，序列 (X3(ω), X4(ω), ...) 是相对紧的，且其极限点集合恰好是 K。

含义: 与Donsker定理不同（那里 √nXn1 ⇒ N），Strassen定理用更小的标准化因子，使得标准化后的极限行为不再是"收敛到一点"，而是"在紧集 K 中徘徊"。这推广了经典的Hartman-Wintner重对数律。

证明策略

第一步: 对Brownian运动证明类似的结果(Theorem 22.2)：对 Bnt = B(nt)/√(2n log log n)，以概率1，序列相对紧且极限点集合等于 K。

该证明分两部分: (a) 证明所有极限点在 K 中（通过 χ2 分布的尾概率和差分的精细估计）；(b) 证明 K 中的每个点都是极限点（通过Brownian运动增量的独立性构造使得"有利事件"无限次发生）。

第二步: 利用Theorem 20.1中Skorohod嵌入给出的关系 ||Yn − Bn|| → 0（但这里使用 Ln = √(2n log log n) 标准化），将Brownian运动的结果转移到原始部分和过程。

Lemma 1. 集合K的性质

K 是紧的: 由Arzela-Ascoli，因为 |x(t)−x(s)| ≤ ∫st |x'(u)|du ≤ √(t−s)（Schwarz不等式），所以 K 中的函数一致等度连续且一致有界。闭性的证明用到 L2 中单位球的弱紧性。

K 也是凸的，但在 C 中是nowhere dense的（没有内部点）——它更像平面中的闭线段而不是闭圆盘。

Example 22.1 (Hartman-Wintner定理)

取 φ(x) = x(1)（连续泛函）。由Lemma 2，极限点集合是 φ(K) = [−1, 1]（因为 x(t)=t 和 x(t)=−t 都在 K 中）。因此:

limsupn Sn/√(2n log log n) = 1,   liminfn = −1

这是经典的上极限重对数律。

Example 22.2 (上确界泛函)

取 φ(x) = supt x(t)，则 φ(K) = [0,1]。极限点填满 [0,1]。

Example 22.3 (加权积分泛函)

取 φ(x) = ∫01 x(t)f(t)dt，其中 f 有有界导数。这等价于 L2 中的内积 ∫ F(t)x'(t)dt。在约束 ∫ (x')2 ≤ 1 下，最大值是 m = ||F||2。因此 φ(K) = [−m, m]。

M1-M10: Metric Spaces / 度量空间

M1 基本记号

度量空间 (S, ρ) 由空间 S 和度量 ρ(x,y) 组成。对于子集 A ⊆ S: 闭包 A−，内部 A◦，边界 ∂A = A− ∖ A◦。距离 ρ(x,A) = inf{ρ(x,y): y ∈ A}。开球 B(x,r) = [y: ρ(x,y) < r]。ε-邻域 Aε = [x: ρ(x,A) < ε]。

M2 比较度量

两个度量 ρ 和 ρ' 在同一个空间上。ρ'-拓扑比 ρ-拓扑更细，若 O ⊂ O'，等价于 ρ'(xn,x) → 0 ⇒ ρ(xn,x) → 0。恒等映射 i: (S,ρ') → (S,ρ) 连续当且仅当此条件成立。离散度量: ρ(x,y) = 1 若 x ≠ y。

M3 可分性: 三个等价条件

以下三个条件等价: (i) S 可分（存在可数稠密子集）；(ii) S 有可数基；(iii) S 的每个子集的每个开覆盖有可数子覆盖。
子集 M 可分意味着存在 M 中的可数稠密子集。
林德洛夫性质(Lindelöf property): 可分子集的每个开覆盖有可数子覆盖。可分性是拓扑性质（在等价度量下保持不变）。

M4 完备性

序列 {xn} 是基本列(Cauchy列)若 supi,j>n ρ(xi, xj) → 0。集合 M 完备若 M 中每个基本列有极限在 M 中。有界度量 ʀ(x,y) = 1 ∧ ρ(x,y) 与 ρ 等价，且一个空间是 ρ-完备的当且仅当它是 ʀ-完备的。完备性不是拓扑性质（[1,∞)完备但同胚的 (0,1] 不完备），但空间可以是拓扑完备的（存在等价度量使其完备）。

M5 紧性: 三个等价条件

以下三个条件等价: (i) A− 紧（每个开覆盖有有限子覆盖）；(ii) A 中的每个序列有收敛子序列（极限在 A− 中）；(iii) A 是全有界的（对每个正 ε 有有限 ε-网）且 A− 完备。

重要事实: 紧集的连续像是紧集。在欧氏空间 ℝk 中，有界性等价于全有界性，因此有界闭集就是紧集。

Original - M6 Products of Metric Spaces

If each Si is separable, then the product is separable. If each Si is complete, then the product is complete. If Ai is compact in Si, then A1 × A2 × ... is compact in the product (Tihonov's theorem, special case for metric spaces proved via the diagonal method).

M7 Baire Category: If S is complete, it cannot be represented as a countable union of nowhere dense sets. A set of the first category is a countable union of nowhere-dense sets; otherwise it is of the second category.

M8 Upper Semicontinuity & Dini's Theorem: f is upper semicontinuous iff [x: f(x) < a] is open for each a. Dini's theorem: If fn(x) ↓ 0 and each fn is upper semicontinuous, the convergence is uniform on each compact set.

M9 Lipschitz Functions: If |f(x)−f(y)| ≤ Kρ(x,y) on A, then f can be extended to all of S with the same constant: g(x) = infy∈A(f(y) + Kρ(x,y)).

M10 Topology and Measurability: Continuous maps are Borel measurable. Pointwise limits of measurable maps are measurable. The set Dh of discontinuities of any map h: S → S' is a Borel set.

For separable product spaces, the Borel σ-field equals the product σ-field: S' × S'' = T.

M6 乘积空间: 每个 Si 可分则乘积可分；每个 Si 完备则乘积完备；紧集乘积紧（度量空间情形下由对角线方法证明）。

M7 Baire范畴: 完备空间不能表示为可数个无处稠密集的并。第一范畴集: 可数个无处稠密集的并；否则为第二范畴。

M8 上半连续性与Dini定理: f 上半连续当且仅当 [x: f(x) < a] 对每个 a 是开集。Dini定理: 若 fn(x) ↓ 0 每个 fn 上半连续，则在每个紧集上收敛是一致的。

M9 Lipschitz函数延拓: 满足 |f(x)−f(y)| ≤ Kρ(x,y) 在 A 上的函数可以延拓到全空间，保持相同的Lipschitz常数。

M11-M16: Analysis / 分析

M11 Gamma函数与Beta函数

Gamma函数: Γ(a) = ∫0∞ xa−1 e−x dx。满足 Γ(a) = (a−1)Γ(a−1)，所以 Γ(m) = (m−1)!。

Gamma分布: 密度 ga(x) = xa−1e−x/Γ(a) 在 (0,∞) 上。

Beta函数: B(a,b) = ∫01 ya−1(1−y)b−1 dy = Γ(a)Γ(b)/Γ(a+b)。Beta分布: ya−1(1−y)b−1Γ(a+b)/Γ(a)Γ(b) 在 (0,1) 上。

M12 Dirichlet公式

令 Dk 为满足 t1, ..., tk > 0 且 Σ ti < 1 的 ℝk 区域。Dirichlet积分公式:

∫Dk t1α1−1 ... tkαk−1 (1−Σ ti)αk+1−1 dt1...dtk = Γ(α1)...Γ(αk+1)/Γ(α1+...+αk+1)

M13 Dirichlet分布

随机向量 X = (X1, ..., Xk) 具有(正)参数 α1, ..., αk 的Dirichlet分布，若 Xk = 1 − Σi=1k−1 Xi 且 (X1, ..., Xk−1) 在单形上有Dirichlet密度。对称Dirichlet分布: 所有参数相等，分布在分量置换下不变。此分布在§4（GEM和Poisson-Dirichlet分布）中起核心作用。

Original - M14 Scheffe's Theorem

If gnm and gm are nonnegative, Σ gnm = Σ gm (finite), and gnm → gm for all m, then Σ |gnm − gm| → 0. This is used to show that if the discrete distributions converge, then Pnf → P f for bounded continuous f.

M15 Measurability of Some Mappings: The mapping (x, t) ↦ x(t) from C × [0,1] to ℝ is measurable. The function h(x) = ∫ v(x(t)) dt (where v is Borel measurable, bounded, and continuous except on a set of Lebesgue measure 0) is measurable C and continuous except on a set of Wiener measure 0. The supremum map h(x) of t where x(t) = 0 is also shown to be measurable and continuous W-almost surely.

M16 More Measurability: The mapping ψ: D × D0 → D defined by ψ(x,φ) = x ∘ φ is measurable D × D0/D. The proof uses the finite-dimensional sets in D.

M17-M18: Convexity / 凸性

Original (Cleaned)

We treat a few facts about planar convex sets needed for Section 15. Let T be the space of compact convex subsets of Q = [0,1]2. The Hausdorff distance: h(A,B) = (supx∈A infy∈B |x−y|) ∨ (supy∈B infx∈A |x−y|).

M17: T is h-complete: if {Cn} is h-fundamental, define C = ∩n(∪i≥n Ci)−; then C is convex and h(Cn,C) → 0.

Key estimate on the boundary: λ(Cε) − λ(C) = l(∂C)ε + πε2 if C◦ ≠ ∅, where λ is 2D Lebesgue measure and l is arc length. Also, l(∂C) ≤ 9 if C ⊂ Q.

A boundary point of a convex set is regular if through it runs only one support line. Every convex set can be approximated arbitrarily closely in the Hausdorff metric by a larger convex set with all regular boundary points and nonempty interior.

此部分处理第15节（经验过程的凸集索引）所需的平面凸集知识。

M17 Hausdorff距离与凸集的完备性: T（单位正方形内紧凸集的全体）在Hausdorff距离下是完备的。如果 {Cn} 是 h-基本列，则 C = ∩n(∪i≥n Ci)− 是凸集且 h(Cn, C) → 0。

M18 ε-网的尺寸（度量熵估计）

存在常数 A 使得对 0 < ε < 1，T 中存在由 N(ε) 个凸多边形组成的 ε-网 T(ε)，且:

log N(ε) ≤ A ε−1 log(1/ε)

更进一步，每个 C ∈ T 可以从上方被逼近: 存在 C' ∈ T(ε) 使得 C ⊂ C' 且 h(C, C') < ε。

证明思路: 用有限个方向（阶梯 ∼ 1/√ε 条射线）上的支撑线构造多边形逼近，再用格点量化顶点位置。Dudley对此不等式有更精细的版本（可移除对数因子）。

M19-M23: Probability / 概率论

M19 Etemadi不等式

若 S1, ..., Sn 是独立随机变量的部分和，则:

P[maxk≤n |Sk| ≥ 3α] ≤ 3 maxk≤n P[|Sk| ≥ α]

这是Kolmogorov不等式的一个推广，不需要同分布假设，且使用更灵活。

M20 Bernstein不等式

若 η1, ..., ηn 是独立随机变量，均值为0，方差为 σ2，每个有界于1。令 S = Σ ηi，s2 = nσ2。则:

P[|S| ≥ x√n] ≤ 2 exp[−x2/(2σ2 + 2x/3√n)]

证明方法: 利用矩母函数 E[etη] ≤ exp[σ2t2G(t)/2] 其中 G(t) = 1/(1−t)，然后优化 t。这是Hoeffding不等式的一般形式，提供了亚高斯衰减。

M21 Renyi混合

事件序列 An 是Renyi混合的如果: P(An ∩ E) → αP(E) 对每个 E ∈ F 成立。此时 P(An) → α。

定理(推广): 如果该条件对域 F0 中每个 E 成立，则它实际上对 F = σ(F0) 成立（通过λ-π定理和条件期望论证）。

这个结果确保了: 若 P0 被 P 控制，则 P0(An) → α 也成立。

M22 移位算子与平稳过程的表示

设 R±∞ 是双向无穷实数序列空间。平移算子 T: ξn(Tx) = ξn+1(x)。

任何平稳过程都可以表示在 (R±∞, R±∞, P) 上: P T−1 = P（平移不变）。过程是遍历的当且仅当 T 在 P 下是遍历的。

单向平稳过程可以延拓为双向平稳过程。对于鞅差情形: 若 E[ξn|ξ1,...,ξn−1] = 0，则双向版本满足 E[ξ1|ξk,...,ξ−1] = 0。

关键公式: E[θ∘T | T−1F] = E[θ|F] ∘ T（移位下的条件期望）。

M23 p-混合 (p-mixing)

对于平稳过程，度量最近与最远的过去和未来之间的相关性:

ρn = sup |E[ξη]|,   ξ∈Fk, η∈Gk+n, Eξ=Eη=0, ||ξ||≤1, ||η||≤1

基本不等式[M23(48)]: |E[ξη(Tnx)]| ≤ ρn ||ξ|| · ||η|| 对于 ξ ∈ F0 和 η 有界可测。

这与§19中混合过程的CLT条件 Σ ρn < ∞ 直接相关。

总结 / Summary

本文件覆盖了Billingsley Convergence of Probability Measures (1999) 第4-5章及附录M的全部核心内容:

  • Ch4 (相依变量): 从素数因子（算术模型）到鞅到一般平稳过程，层层递进地展示了相依条件下的极限理论。核心思想: 如果依赖"衰减得足够快"，则极限行为与独立情形相同（正态、Poisson、Brownian、Poisson过程）。
  • Ch5 (其他收敛模式): 三种加强Donsker定理的构造: Skorohod嵌入（轨道级逼近，20节）、正态序列耦合（分块构造+Prohorov距离，21节）、Strassen重对数律（几乎必然的相对紧性，22节）。
  • Appendix M: 全书的技术附录，涵盖度量空间基础知识（可分性、完备性、紧性）、分析工具（Gamma函数、Dirichlet分布）、凸性（Hausdorff距离、熵估计）、概率不等式（Etemadi、Bernstein）和平稳过程的形式化。

本书被认为是弱收敛理论的经典之作，其独特的"逐步具体化"的写作风格——从最一般的度量空间理论逐步收敛到 C[0,1] 和 D[0,1] 上的具体极限定理——使其成为概率论研究生教育的标准教材。
